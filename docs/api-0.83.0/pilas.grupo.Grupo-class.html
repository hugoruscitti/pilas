Class Grupo

source code

Un grupo es un contenedor que funciona como una lista normal, pero mejorada.

Los grupos pueden contener actores, y permite que a todos los actores se los pueda tratar como uno.

Por ejemplo si tienes un contenedor con 20 actores, podrías ampliar el tamaño de todos ellos juntos usando la sentencia:

>>> grupo = pilas.atajos.fabricar(pilas.actores.Mono, 20)
>>> grupo.escala = 2
